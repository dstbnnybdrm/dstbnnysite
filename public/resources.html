resources

a collection of applications and resources that i enjoy and use! (constantly updating!)

music creation

  • DAWs that i use
    • ableton: my main DAW these days
    • fl studio: used primarily for ~5 years
  • music trackers i've used
    • fasttracker II clone: what i've been using most lately, actually
    • renoise: if you want a more DAW-like experience
    • milkytracker: probably what i would tell someone new to trackers to start with
  • vst plugins
    • tweakbench plugins: lovely lofi and sometimes strange, plugins. i use some of these quite frequently! (only available in 32bit, so i use jbridge to be able to open them in ableton)
    • analog obsession plugins: incredible analog-style plugins, all for free!
    • codec by lese: super cool and dirty digital degradation; i adore this sound. free too!
  • sample packs
    • seaweed factory sample packs: lots of toy keyboards, well-known synths, and spacey impulse responses, i use these a lot!
    • studio brootle sample packs : lots of great free drum machine sounds!
    • and of course: some shameless self promotion :)

music listening

  • musicbee: used mainly for it's scanning, organising, and tagging features
  • winamp 5.666: my main music player lately, love the simplicity and having cute skins!

code

  • Neovim: my IDE/code editor (check here for my config :3c)
  • fish: the shell i use
  • Conventional Commits: simple and useful Git commit message convention

web

  • deploy-to-neocities: what i use to deploy this site!
  • BEM: the CSS methodology i use

    • took some pointers from the BEM cheat sheet, super concise and helpful!

  • CSS-tricks: a great abundance of CSS information!
  • this site loosely follows the tenets of brutalist design
  • Accessibility Developer Guide: valuable info on making your site more accessible!
  • a great article on spacing components in CSS!
  • super helpful guides on typography!:
    • Butterick's Practical Typography
    • Harmonious Typography and Grid

general

  • Dvorak: keyboard layout i use for several reasons
  • FancyWM: dynamic tiling window manager for Windows
