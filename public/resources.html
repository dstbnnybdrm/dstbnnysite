resources

a collection of applications and resources that i enjoy and use! (constantly updating!)

music creation

  • DAWs that i use
    • ableton: my main DAW these days
    • fl studio: used primarily for ~5 years
  • music trackers i've used
    • renoise: my main tracker, provides a more DAW-like experience
    • fasttracker II (FT2) clone: fun little clone of a classic
    • milkytracker: free and modern FT2 inspired tracker
  • vst plugins
    • tweakbench plugins: lovely lofi and sometimes strange, plugins. i use some of these quite frequently! (only available in 32bit, so i use jbridge to be able to open them in ableton)
    • analog obsession plugins: incredible analog-style plugins, all for free!
    • codec by lese: super cool and dirty digital degradation; i adore this sound. free too!
  • sample packs
    • seaweed factory sample packs: lots of toy keyboards, well-known synths, and spacey impulse responses, i use these a lot!
    • studio brootle sample packs : lots of great free drum machine sounds!
    • goldbaby free packs: bunch of recordings of gear, synths, and drum machines!
    • and of course: some shameless self promotion :)

music listening

  • musicbee: wonderful player, i love it especially for its scanning, organising, and tagging features
  • winamp 5.666: simple and classic player, you can find a bunch of fun skins here!

code

  • Neovim: my IDE/code editor (check here for my config :3c)
  • Zsh: the shell i use, alongside Oh My Zsh
  • Conventional Commits: simple and useful Git commit message convention

web

  • BEM: the CSS methodology i use

    • Butterick's Practical Typography

  • File Garden: simple, no frills file hosting site, i use it to host audio files and images!
