music production

DAWs & software

Ableton
my main DAW these days
FL Studio
DAW i used to use primarily for ~5 years, though i still occasionally open it up as a VST
Renoise
my main tracker, i like its workflow more than a traditional DAW for some genres/styles
VCV Rack 2
free virtual modular synth platform, really fun stuff once you get your mind around it, i like making nonsense ambient stuff with it
SynthV
free vocal synthesizer (in the vein of VOCALOID and such), just started messing around with it and it's been very fun!
MiniMeters
super simple but very helpful and sleek looking audio metering app! i use it all the time

plugins

Melodyne
pitch correction software
Analog Obsession plugins
incredible analog-style plugins, all for free! what i use most often are the FETISH, VariMoon, and dBComp compressors, the BritPre pre-amp, and the FrankCS channel strip!
LA-2A compressor
got this in a giveaway they were doing for a bit, it's a classic compressor for a reason! love putting it on vocals and acoustic guitars
MeldaProduction plugins (the free FX bundle)
super useful set of free plugins! i specifically use the MAutoPitch plugin all the time for formant shifting and that exaggerated autotune sound!
Tweakbench plugins
really charming bunch of weird and lofi plugins. i use some of these quite frequently! (only available in 32 bit, so i use jBridge to be able to open them in ableton)
SketchCassette II
cassette tape-inspired degradation, i love love love cassettes and this is possibly the best and most accurate cassette tape emulation ever!
Codec
super cool and dirty digital degradation; i adore this sound. free too!

code

Neovim
my IDE/code editor (check here for my config :3c)
Zsh
the shell i use, alongside Oh My Zsh
Windows Terminal
my terminal (yes i use Windows, i don't wanna hear anything about it! lol)
tmux
terminal multiplexer, lets me open multiple windows in one session, makes me feel like a real programmer.~
FiraCode
the font i use for my terminal, specifically the patched Nerd Font version

web

deploy-to-neocities
what i use to deploy this site!
Sass
CSS extension language i use and has become essential to my workflow!
File Garden
simple, no frills file hosting site, i use it to host audio files and images!

general

Obsidian.md
markdown-based note-taking app!
Thunderbird
free and open-source email client! also has a built-in RSS feed reader!
FancyWM
dynamic tiling window manager for Windows
PowerToys
bunch of utilities to make Windows more bearable to use!
Dvorak
keyboard layout i use! (here's some reasons why!)
AutoHotKey
software and scripting language to create macros, shortcuts, hotkeys, etc., i use it to emulate parts of an HHKB keyboard lol
f.lux
adjusts the temperature of your display to adapt to the time of day! helps me reduce eye strain since i look at screens all day

music listening

  • musicbee: wonderful player, i love it especially for its scanning, organising, and tagging features
  • WACUP: i.e. the WinAmp Community Update Project, a simple and very customizable player based off of a classic! you can find a bunch of fun skins here!

hardware/gear

general peripherals

Focusrite Scarlett 4i4
my audio interface! (i specifically use the 3rd generation version)
Wacom Intuos Pro
the (very old) drawing tablet i use! i actually don't use a mouse much if ever anymore unless i'm playing games, every other time i use this tablet
Arturia MiniLab MkII
the MIDI keyboard i use!

speakers/headphones

Yamaha HS5s
studio monitors i use!
Beyerdynamic DT 990 Pro
studio headphones i use! (i somehow broke the fucking cable i had to jankily solder on a new knockoff one lmao)
Grado SR80x

microphones

Shure SM57
super reliable dynamic microphone! the only studio quality mic i have right now so i use it for everything lol
Blue Yeti
USB condenser microphone, i used it when i didn't have any other options and it worked fine at the time! i still occasionally use it when i intentionally want the specific quality that it has
WOW! Fiesta karaoke mic
sometimes i like to use this very shitty mic i found in an old karaoke machine set we have lol. it uses a 1/4" jack instead of an XLR so i like doing silly things like putting it through my amp and stuff

cassette recorders

QFX Retro-39 Shoebox Recorder
relatively cheap cassette recorder i got online, it's nothing too amazing but it definitely does its job well enough! i mostly use it to record with its onboard mic
Optimus CTR-117
surprisingly decent sounding cassette recorder i managed to thrift! its onboard mic is finicky so i mostly use it to record from my PC to cassette
modded Byron Static player
probably the cheapest cassette player/recorder i could find online at the time. i did a pretty easy and common motor speed control mod to it.
